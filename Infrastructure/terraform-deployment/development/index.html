S3WWW Project

Static Website Hosting with S3-Compatible Storage

Welcome to S3WWW

S3WWW is a powerful and lightweight solution for hosting static websites using S3-compatible storage backends. This project provides a seamless way to serve web content directly from object storage with high availability and scalability.

Features:

  • Fast Deployment - Quick setup with Docker and Kubernetes support
  • Cloud Native - Works with AWS S3 and minio and other S3-compatible storage
  • Configurable - Flexible configuration for different environments
  • Monitoring - Built-in health checks and monitoring capabilities
Environment: Development

S3WWW Project. Built with love for modern web hosting.

Built with Go. Deployed with Helm
